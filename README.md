## Problem

Postman removed their free tier and developers need a truly local-first alternative without pricing concerns or account requirements

## Who Is This For?

Web developers and API testers who want privacy and ownership of their data

## Features

- **Offline request builder with environment variables and history**
- **Local collection storage with JSON import/export**
- **Response viewer with syntax highlighting and formatting**
- **Zero-setup installation running entirely in the browser**

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/malikmuhammadsaadshafiq-dev/mvp-localapi.git
cd mvp-localapi
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
